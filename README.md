# 🕵️ تطبيق الدردشة السرية (Secret Chat App)

تطبيق دردشة آمن تماماً، لا يحتفظ بأي سجلات (Zero-Logs)، ويستخدم التشفير من طرف إلى طرف (End-to-End Encryption).

## 🚀 المميزات
- **بدون قاعدة بيانات:** بمجرد إغلاق التطبيق، تختفي الرسائل للأبد.
- **تشفير AES-256:** يتم تشفير الرسائل في متصفحك، ولا يمكن للخادم قراءتها.
- **غرف خاصة:** يمكنك إنشاء غرف بكلمات مرور خاصة للتشفير.
